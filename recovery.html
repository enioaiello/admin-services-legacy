Mode de récupération

Si Admin Services ne se comporte pas correctement, que votre profil est inaccessible ou que vous avez des problèmes avec le système de redirection, vous pouvez diagnostiquer et réparer Admin Services via ce menu.

  • Réinitialiser le profil
  • Mettre à jour le profil
  • Réinitialiser Admin Services
  • Réinitialiser le système de démarrage
  • Démarrer Admin Utility (sans échec)
  • Télécharger la dernière mise à jour stable via GitHub
  • Quitter et passer sur Admin Services

Vous pouvez utiliser ces options pré-enregistrées afin de rendre Admin Services utilisable.

Si vous avez installé Admin Services sur un service tiers (ou en local), vous pouvez télécharger une copie à jour. Cela règle les problèmes de corruption.

Vous pouvez également ouvrir la console (au format ordinateur) pour effectuer des actions précises.

Si cela est une erreur, vous pouvez quitter et passer sur Admin Services (si vous avez modifié la redirection pour démarrer google-fake-admin-panel, par exemple, quitter continuera la redirection).

Réinitialiser le profil ?

Cela effacera toutes vos informations personnelles, telles que votre nom d'utilisateur ou votre mot de passe.

  • Oui
  • Non

Mettre à jour le profil ?

Cela mettra à jour le compte, tout en conservant vos informations de profil.

  • Oui
  • Non

Réinitialiser Admin Services ?

Cela tentera de restaurer les paramètres tout en conservant votre version actuelle d'Admin Services.

  • Oui
  • Non

Réinitialiser le système de démarrage ?

Cela effacera les redirections vers google-fake-admin-panel ou des redirections externes et rétablira la redirection vers Admin Services (si un compte est connecté, Fast-reconnect sera actif).

  • Oui
  • Non

Démarrer Admin Utility (sans échec)

Cela démarrera Admin Utility sur le domaine enioaiello.github.io.

Si vous hébergez Admin Services sur votre serveur, téléchargez Admin Utility depuis GitHub.

Remarque : à compter du 20 janvier 2025, il n'est plus possible de télécharger Admin Utility.

  • Retourner à l'accueil

Télécharger la dernière mise à jour stable via GitHub

Cela vous proposera de télécharger le dernier commit (ou un commit personnalisé) sur le dépôt GitHub officiel.

Vous devrez installer et configurer chaque fichier à la main ou les remplacer.
