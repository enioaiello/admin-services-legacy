Avenir du service

Chers utilisateurs d'Admin Services,

Admin Services vient de bénéficier sa dernière mise à jour de fonctionnalité.

Qu'est-ce que cela signifie ?

Cette version d'Admin Services ne bénéficiera pas de fonctionnalités supplémentaires.

Les seules mises à jour qui seront disponible seront :

  • Ajout, modification et suppression d'application
  • Correction de bugs

Cette décision fait suite à une nouvelle version majeure de ce projet : le passage en PHP.

Une réécriture totale est planifiée, permettant de faire un grand ménage.

Cette nouvelle version bénéficiera de base de données, de nouvelles fonctionnalités, d'un nouveau design plus agréable, d'une correction total du code (bonnes pratiques, optimisation, vie privée, sécurité, orthographe…).

Quoi de neuf dans cette mise à jour ?

  • Un nouveau logo est disponible
  • Une nouvelle page d'accueil plus simple
  • Nouvelle version mobile
  • Le bouton d'accueil de la barre de navigation est déplacé dans le logo
  • Les fautes d'orthographes ont été corrigées
  • Quelques changements mineurs dans le code pour simplifier sa maintenance

Trois fonctionnalités retirées : OOBE, mode sans échec, admin-utility.
