Effacer votre compte

Attention : La suppression de votre compte supprimera l'entièreté de vos informations. Il vous faudra donc peut-être recréer un compte pour accéder à Admin Services.

Il vous faudra pour cette opération :

  • Votre adresse email
  • Votre mot de passe

Effacer votre compte supprimera les informations suivantes :

  • Information de personnalisation d'Admin Services
  • Information de compte utilisateur
  • Accès aux fonctionnalités expérimentales

Vous ne pourrez pas récupérer votre compte.
