Intégration d'applications interne

Bonjour.

Suite à la version 6.0.0 d'Admin Services, une nouvelle application est apparue sur Admin Services.

Utilisez des applications officielles ou de développeurs tiers directement sur Admin Services.

Accéder à une application

Sur votre page d'accueil, descendez jusqu'à atteindre le menu Applications, qui se trouve après la personnalisation de votre page d'accueil.

Vous pouvez, à partir de ce menu, lancer une application officielle ou tierce recommandée, ou accéder à toutes les applications vient un simple bouton.

Soumettre une application

Vous devrez contacter Enio Aiello afin que vous lui communiquez votre idée d'application.

Gardez cependant contact, si l'idée est intéressante, vous serez recontactés afin d'obtenir un kit de développement qui contient les règles de l'applications.

Assurez-vous de rendre compatible votre application avec GitHub Pages, donc il ne faut que de l'HTML, du CSS, du JS et toute autre technologie prises en charges par ce service.

Si votre application est jugée intéressante, elle sera publiée dans l'accueil de l'utilisateur, ou dans la liste des applications.
