Mise à jour majeure

Bonjour.

La version 8.0.0 arrivera le 1er Novembre 2023. Cette mise à jour sera une refonte intégrale du site.

Voici les nouveeautés :

  • Reprise rapide de session
  • Nouvel écran de connexion
  • Refonte totale du site
  • Nouvelle interface graphique : ArticlesUI
  • Tri dans les localStorage
  • Nouvelle sécurité
  • Nouveau centre de sécurité
  • Modifications du centre d'applications
  • Certaines pages sont désormais dans un dossier à part (applications) avec les applications des développeurs
  • Simplification du code afin que certaines actions soient automatique
  • Corrections de toutes les erreurs W3C
  • Corrections des erreurs de frappe
  • Nouvelle page d'accueil
  • Nouveaux paramètres
  • Corundum ne participera plus à Admin Services

Certaines fonctionnalités du site seront temporairement retirées et seront réintégrés avec le functionPack1.

Cette mise à jour sera déployée gratuitement pour tous les utilisateurs du service.

Les développeurs Corundum, les développeurs d'applications, les membres bêta et alpha ont déjà reçu la developper preview 1 d'Admin Services 8.

Calendrier

21 Octobre : Lancement de la developper preview 1 pour les utilisateurs alpha et les développeurs Corundum.

1er Novembre : Lancement d'Admin Services 8 stable pour tous les utilisateurs.

2 Novembre : Début de la mise à jour vers les functionsPack1 (les membres bêta recevront 1 mise à jour toutes les semaines).
