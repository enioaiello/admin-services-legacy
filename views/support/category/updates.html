Disponibilité du support

Comme pour chaque logiciel important, tels que des systèmes d'exploitation, Admin Services contient un système de support particulier. Étant un site Web, il n'y a pas plusieurs versions disponibles sur navigateur. Cependant, vous pouvezdowngrade vers une version antérieure via GitHub.

Le support est donc proposé systématiquement pour deux versions, les versions actuelles et la version antérieure à celle-ci.

Ce système de support ne garantit pas de mises à jour supplémentaires pour les anciennes versions, mais permet de porter assistance via le système de support ou par email.

Voici la disponibilité de support pour Admin Services (cela ne s'applique pas pour les versions customisées) :

  • Version 10.x.x (actuelle) : Support disponible
  • Version 9.0.x : Support disponible
  • Version 8.x.0 : Support non disponible
  • Version 7.x.0 : Support non disponible
  • Version 6.x.x : Support non disponible
  • Version 5.0.0 : Support non disponible
  • Version 4.x.x : Support non disponible
  • Version 3.0.x : Support non disponible
  • Version 2.0.x : Support non disponible
  • Version 1.0.0 : Support non disponible
  • Versions alpha et canal développement : Support non disponible

Concernant l'application de bureau, qui a été supprimée, aucune version n'est supportée à l'heure actuelle, nous vous conseillons également de supprimer Admin Services Web de votre poste de travail.

Veuillez également noter que les croix présentes dans les numéros de versions indiquent que cela concerne toutes les mises à jour postées pour cette version.

Versions modifiées

Vous pouvez télécharger Admin Services et apporter des modifications en interne ou pour plusieurs postes, cependant, celles-ci ne sont pas contrôlées, et peuvent donc ne plus être supportées officiellement, contenir du code malveillant, contenir des bugs non résolus, contenir d'anciennes versions de paquets (tels qu'ArticlesUI).

Certaines fonctionnalités liées aux versions modifiées peuvent également être différentes du système de support ou d'astuces.
