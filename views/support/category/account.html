Connexion

Vous avez du mal à vous inscrire ou vous connecter ?

Si vous avez un compte enregistré, utilisez-le dans Fast-reconnect (par défaut), ou dans la page de connexion, si Fast-reconnect n'est pas activé.

Si c'est la première fois que vous utilisez Admin Services, choisissez l'option de connexion d'inscription, ou dans le cas où vous êtes développeur, connectez-vous avec votre identifiant.

Lorsque vous sélectionnez l'étape d'inscription à partir d'Admin Services 10, un nouveau système d'inscription plus complet se lance : l'OOBE. Vous pourrez alors configurer votre photo de profil, votre nom d'utilisateur, votre adresse email, votre mot de passe, votre fond d'écran et les paramètres de sécurité liés à la connexion plus rapidement.

À savoir qu'aucun email ne sera envoyé durant toute votre inscription, mais il servira à ajouter une sécurité supplémentaire sur votre compte, vous pouvez donc mettre n'importe quelle adresse email.

Mot de passe

Vous avez perdu votre mot de passe ?

Si vous avez perdu votre mot de passe, vous pouvez le réinitialiser de plusieurs manières, à savoir que certaines ne sont plus compatibles à partir d'Admin Services 10 si vous avez modifié les paramètres de sécurité.

Admin Services 10 et antérieur

Vous pouvez réinitialiser votre mot de passe grâce aux outils internes d'Admin Services, tel que le système de récupération, ou si Fast-reconnect est actif, via l'option de réinitialisation de mot de passe.

Si vous modifiez le mot de passe depuis les localStorage, attention !

Si les systèmes de sécurité sont actifs (par défaut), modifier votre mot de passe de cette façon pourrait vous faire perdre l'accès à votre compte.

Vérifiez donc si les sauvegardes de données sont disponibles, si elles le sont, ne modifiez pas sa valeur.

Admin Services 9 et antérieur

Vous pouvez utiliser Admin Utility, présent par défaut à partir d'Admin Services 8, ou en téléchargement libre ici (si vous téléchargez Admin Utility, vous devrez avoir téléchargé Admin Services).

Ouvrez Admin Utility et modifier votre mot de passe via cette interface.

Autres solutions pour Admin Services 9 et antérieur

Vous pouvez directement modifier les localStorage via une console de développement. Voici une commande qui marche sur la plupart des versions :

localStorage.setItem('password', 'votre_mot_de_passe');

Effacer votre compte

Vous voulez effacer votre compte ?

Si vous voulez effacer votre compte, vous pouvez le faire via l'interface de Admin Services, ou via Admin Utility.

Via le centre de sécurité

Accédez à cette page du centre de sécurité.

Réinitialisez ensuite votre compte.

Via le système de récupération (à partir d'Admin Services 10)

Accédez aux paramètres locaux et rechargez Admin Services en mode de récupération. Choisissez ensuite l'option de réinitialisation d'Admin Services.

Via Admin Utility (recommandé en dessous d'Admin Services 8)

Accédez à l'interface graphique d'Admin Utility, ignorez l'avertissement et descendez jusqu'à voir le bouton de réinitialisation.

Cliquez et quittez Admin Utility.
