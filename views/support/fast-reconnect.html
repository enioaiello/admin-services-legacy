Fast-reconnect

Avec la version 9.0.0 d'Admin Services, les utilisateurs peuvent se reconnecter plus rapidement.

Cette fonctionnalité est activée mais vous pourrez bientôt la désactivée.

Comment ça marche ?

Il vous suffit de vous connecter normalement, puis de quitter la page. Lorsque vous reviendrez, vous tomberez sur cette nouvelle page.

Cette nouvelle page contient le moins d'information possible afin que vous puissiez vous reconnecter sans prise de tête.

Cette page s'adapte entièrement au thème de votre navigateur, comme l'intégralité des pages.

Quel composant prend le relais ?

Comme vous l'avez vu avec la 8.0.0, une nouvelle interface à été introduite, ArticlesUI. Cette interface, faites pour être simple d'utilisation et accessible de tous, gère maintenant désormais entièrement cette page.
