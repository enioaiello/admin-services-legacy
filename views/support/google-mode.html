Activation du Google Mode

Qu'est-ce que le Google Mode ?

Le Google Mode, permet de charger le Google Fake Admin Panel, qui avait disparu après la création d'Admin Services.

Le Google Mode permet d'afficher, de manière stable, la dernière version sortie de Google Fake Admin Panel.

Cette dernière a été adaptée pour Admin Services.

Comment ça fonctionne ?

Pour utiliser Google Fake Admin Panel, vous aurez besoins d'accéder aux paramètres.

Cherchez ensuite les paramètres d'Admin Services et cliquez sur rétrograder.

Identifiant de connexion

Google Fake Admin Panel ne permet pas la création de comptes.

Il vous faut donc le compte universel. Ce dernier permet d'accéder à toutes les fonctionnalités de Google Fake Admin Panel, de manière temporaire.
