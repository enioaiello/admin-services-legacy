Version 4.0.1

Nouveau design

Le design d'Admin Services est désormais plus simple et accessible à tous. Tout a été changé : la page de connexion, le support et plus !

Nouvelle arborescence

Pour les développeurs qui souhaitent modifier Admin Services, vous avez désormais une meilleure arborescence de fichiers.

Changement de type de comptes

Les comptes n'ont désormais plus de groupe. Et le compte universel a changé. Il sera donc désormais plus simple de vous connectez !

Réadaptation totale

Cette version réadapte le code afin qu'elle puisse être compatible avec l'application officielle.

Thème sombre intégré

Malgré le fait qu'il ne marche pas correctement (si vous avez la solution contactez-nous), le thème sombre fait son apparition !
