Réinitialisation

Admin Services se réinitailise. L'ensemble de vos données sont en suppressions.

La suppression supprime ces données :

  • Nom d'utilisateur
  • Token
  • Email
  • Mot de passe
  • Groupe
  • Fond d'écran
  • Compte SHOPEI
  • Achats SHOPEI

Après la suppression de vos données, vous devrez créer à nouveau un nouveau compte pour utiliser Admin Services ou vous pouvez utiliser le compte universel.

Lorsque cette page sera chargée vous aurez 30 secondes avant la suppression de votre compte. Vous pouvez retourner à l'accueil avant la suppression de votre compte.
