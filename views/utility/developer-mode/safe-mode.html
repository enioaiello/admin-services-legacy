Mode sans échec

Voulez-vous démarrer Admin Services en mode sans échec ?

Démarrer en mode sans échec :

  • Désactivera les redirections au démarrage
  • Désactivera les animations et le style pouvant ralentir, bloquer ou surcharger le navigateur
  • Désactivera les appels à des ressources externes
  • Désactivera certaines applications (dont externe) et les fonctionnalités inutiles
