Vérifier que le mode développeur est actif

Ouvrez la console de développement de votre navigateur. Si vous voyez un avertissement, cela signifie que le mode développeur est actif. Vous pouvez également le voir dans la catégorie "Accueil".

Utiliser les outils

Vous pouvez désormais utiliser les outils recommandés et intégrés. Pour le développement d'une application, téléchargez le SDK (contenant toutes les instructions et trouvable également dans la bibliothèque).
