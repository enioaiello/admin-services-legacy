S'inscrire

Avant de créer un compte, lisez attentivement ce qui suit.

Une fois votre compte créé, il sera appliqué uniquement sur votre appareil, il est donc unique.

Notez aussi que pour supprimer votre compte, vous devenez vous déconnecter.

Si vous vous déconnectez, tout les paramètres liés à votre compte, comme votre token ou votre mot de passe, seront effacés.

Concernant votre compte SHOPEI, nous n'avons pas encore d'informations précises concernant la suppression de votre compte, vous en serez plus prochainement.

Cependant, votre fond d'écran est conservé et le thème sombre.

En continuant, vous acceptez également que votre compte Admin Services soit stocké localement sur votre appareil. Il sera effacé si vous vous déconnectez.

Si vous avez besoins d'assistance, consultez le repo GitHub.
